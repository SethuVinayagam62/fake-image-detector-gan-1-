# AI Fake Image Detection

This project detects AI-generated images using a deep learning model built with TensorFlow and Flask.

## Features
- Upload an image
- Detect AI influence percentage
- Displays prediction result on a web page

## Technologies Used
- Python
- TensorFlow
- Flask
- OpenCV
- HTML/CSS

## How to Run

1. Install dependencies
pip install -r requirements.txt

2. Run the application
python app.py

3. Open browser
http://127.0.0.1:5000/
